靜宜大學中國文學系第八屆畢業成果展—蛻生

蛻，從積澱深處，綻放新生。</p>
與昆蟲不同，哺乳類無法經歷自然意義上的蛻變；然而，人類以「蛻」命名那些無聲卻深刻的心靈成長。</p>
在進入中文系之前，乃至其後，我們皆曾掙扎，在自然與社會的天擇之中匍匐前行。無論選擇何種方向，終將走向屬於自己的羽化時刻。</p>
我們在此凝聚養分，於沉潛中構築外殼，在漫長的時日裡，不斷吸收來自外界的碰撞與滋養，等待時機成熟。</p>
無人能預知毛毛蟲將展現出何種花紋。當旁人的目光尚未停駐之際，我們早已在暗處醞釀，為飛翔做足準備。</p>
為了證明而蛻，為了嚮往而生。</p>
